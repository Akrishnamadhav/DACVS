# DACVS# DACVS – Decentralized Academic Credential Verification System

## 🚀 Overview

DACVS is a blockchain-based system designed to securely issue and verify academic credentials without relying on centralized authorities.

## ❗ Problem It Solves

Fake certificates and manual verification cause fraud and delays.  
DACVS ensures tamper-proof credentials stored on blockchain for instant verification.

## 🛠 Tech Stack

- Solidity (Smart Contracts)
- Node.js & Express
- MongoDB
- Web3 / Ethers.js

## ✅ Current Status

✔ Smart contracts implemented  
✔ Backend API developed  
🚧 Frontend under development

## 🔑 Key Features

- Issue credentials on blockchain
- Verify authenticity publicly
- Tamper-proof storage
- Decentralized trust model
